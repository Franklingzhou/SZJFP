## 一、角色体系（6个角色）

| 角色 | 定位 | 收入来源 | 与阿姨关系 | 诚信档案 | 保证金 | 积分 |
|------|------|---------|-----------|:---:|:---:|:---:|
| **阿姨** | 服务提供者 | 工资（客户付） | 自身 | ✅ | ✅ | ✅ |
| **经纪人** | 撮合成交+维护阿姨+录入阿姨 | 服务费+中介费分成 | 归属人/维护人/录入人 | ✅ | ✅ | ✅ |
| **招生** | 经纪人的阉割版，仅做培训板块 | 培训推荐提成15%+中介费分成 | 录入人 | ✅ | ✅ | ✅ |
| **讲师** | 培训+点评+转简历上户 | 培训授课提成20% | 评价人 | ✅ | ✅ | ✅ |
| **客户** | 服务购买者 | 支付服务费 | — | ✅ | ❌ | ❌ |
| **平台管理员** | 后台管理+审核+结算 | 佣金+培训分成45% | — | ✅ | — | — |

> **⚠️ 经纪人与招生的关系：** 招生是经纪人的功能子集（阉割版），两者共用同一角色系统，通过权限控制区分：
> - **经纪人**：全功能（录入阿姨、一键发单、自动匹配、客户管理、跨经纪人协作等）
> - **招生**：仅培训板块（联系阿姨、推荐培训、录入阿姨转简历上户），**前期不能做发单/匹配等交易业务**
> - 经纪人也可以直接录入阿姨简历上户（与招生路径A一致）
> - 同一人可同时拥有经纪人+招生身份，也可仅作为招生

## 二、产品架构（4端）

### 2.1 阿姨端（小程序）

| 模块 | 功能 | 说明 |
|------|------|------|
| 简历/档案 | 基本信息展示（可分享给客户） | 敏感信息隐藏（手机号/身份证脱敏） |
| 接单大厅 | 浏览经纪人发布的订单 | 按工种/薪资/地区筛选 |
| 上岗状态 | 空闲/在户/待定 | 状态切换，影响接单大厅可见性 |
| 评价中心 | 查看客户评价+经纪人评价+讲师点评 | 三方评价聚合 |
| 诚信档案 | 诚信分+诚信记录 | 违约/投诉/好评等影响诚信分 |
| 保证金 | 缴纳/查看保证金状态和金额 | 平台规定缴纳标准 |
| 培训入口 | 查看可报名培训课程 | Phase 2 |
| 个人中心 | 收入记录、接单历史、中介费支付 | — |

### 2.2 经纪人端（小程序）

| 模块 | 功能 | 说明 |
|------|------|------|
| 入驻认证 | 线上申请→平台审核→入驻 | 身份信息+行业经验+历史业绩 |
| 阿姨管理 | 录入/编辑阿姨档案 | 录入即成为阿姨"归属人"，**经纪人可直接录入阿姨简历上户（同招生路径A）** |
| 阿姨共享库 | 查询匹配其他经纪人的阿姨 | 跨经纪人流通 |
| 一键发单 | 发布客户需求到接单大厅 | 自动匹配系统推荐阿姨 |
| 自动匹配 | 系统根据需求智能推荐阿姨 | 基于工种/经验/状态/评价 |
| 客户管理 | 客户信息+需求记录+订单跟踪 | — |
| 阿姨分享 | 生成阿姨简历分享链接给客户 | 敏感信息自动脱敏 |
| 佣金账单 | 服务费佣金+中介费分成明细 | 月度结算，透明可查 |
| 跨经纪人协作 | 推荐/分享阿姨给其他经纪人 | 触发跨经纪人分配规则 |
| 诚信档案 | 诚信分+诚信记录 | — |
| 保证金 | 缴纳/查看保证金 | — |
| 积分中心 | 积分明细+积分兑换 | 佣金相关活动获取积分 |

### 2.3 招生端（小程序）

> **招生 = 经纪人阉割版**，与经纪人共用同一角色系统，仅展示培训板块功能。

| 模块 | 功能 | 说明 |
|------|------|------|
| 阿姨招募 | 联系阿姨，录入基本信息 | 录入后成为阿姨"归属人"，同经纪人功能 |
| 两条转化路径 | 见下方流程图 | 核心！ |
| 培训推荐 | 推荐阿姨参加培训 | 获取推荐提成15% |
| 阿姨点评 | 对招募的阿姨做点评 | 影响阿姨诚信分和简历 |
| 转简历上户 | 将合格阿姨转入正式简历库 | 触发中介费流 |
| 诚信档案/保证金/积分 | 同经纪人 | — |
| **隐藏功能** | 发单、匹配、客户管理、跨经纪人协作、佣金账单 | **前期招生不可用，后续可按等级开放** |

**招生两条转化路径：**
```
阿姨 → 招生联系
├── 路径A：直接转简历上户（已有经验，无需培训）
│   └── 招生点评 → 转入阿姨共享库 → 经纪人匹配 → 上户
│   └── 收入：中介费分成（招生作为录入人20%）
└── 路径B：需培训合格后转简历上户
    └── 推荐培训 → 讲师培训 → 合格 → 讲师点评 → 转入共享库 → 经纪人匹配 → 上户
    └── 收入：培训推荐提成15% + 中介费分成（招生作为录入人20%）
```

> **与经纪人的区别：** 经纪人也可以直接录入阿姨简历上户（路径A完全一致），但经纪人额外拥有发单、匹配、客户管理、跨经纪人协作等交易功能。前期经纪人不开放培训业务，招生只做培训板块。

## 四、支付与分账体系

### 4.1 资金流向（平台统一收款→分账）

```
客户支付服务费 ──→ 平台统一收款
                      ├── 服务费×80% → 签单经纪人
                      ├── 服务费×8%  → 维护经纪人（跨经纪人场景）
                      └── 服务费×12% → 平台（跨经纪人场景）
                      或
                      ├── 服务费×80% → 签单经纪人
                      └── 服务费×20% → 平台（同经纪人场景）

阿姨支付中介费 ──→ 平台统一收款
                      ├── 中介费×20% → 录入人（招生/经纪人）
                      ├── 中介费×30% → 维护人（经纪人）
                      ├── 中介费×30% → 推荐签单人
                      └── 中介费×20% → 平台

培训费 ──→ 平台统一收款
                      ├── 45% → 平台
                      ├── 20% → 讲师
                      ├── 20% → 谈单人
                      └── 15% → 推荐人（招生）
```

### 4.2 分账方式

| 方式 | 说明 | 适用 |
|------|------|------|
| 微信支付分账 | 平台特约商户，自动分账 | 服务费、中介费 |
| 手工分账 | 平台收款后手动转账 | 培训费（初期） |
| 后台配置 | PC端配置分账比例规则 | 渐进佣金自动切换 |

## 六、诚信档案与诚信分

### 6.1 适用范围：所有角色

| 角色 | 诚信分影响因素 |
|------|---------------|
| 阿姨 | 客户评价、经纪人评价、讲师点评、履约情况、投诉记录 |
| 经纪人 | 成交率、客户满意度、阿姨维护质量、佣金缴纳及时性 |
| 招生 | 录入阿姨质量、培训推荐准确度、阿姨转化率 |
| 讲师 | 培训通过率、学员就业率、点评准确性 |
| 客户 | 付款及时性、评价真实性 |

### 6.2 诚信分规则（初始100分）

| 事件 | 分值变化 |
|------|:---:|
| 获得好评 | +2 |
| 获得差评 | -5 |
| 投诉成立 | -10 |
| 违约/爽约 | -15 |
| 中介费逾期 | -3/次 |
| 诚信分<60 | 冻结接单/发单权限 |
| 诚信分<30 | 退出平台 |

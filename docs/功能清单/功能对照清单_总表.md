## 三、核心业务流程

### 流程1：招生培训线（线索 → 签约 → 培训 → 转简历）

| 步骤 | 说明 | 状态 |
|------|------|:---:|
| 创建线索 | 招生录入姓名+手机号 | ✅ |
| 跟进线索 | 添加跟进记录 | ✅ |
| 签约-发合同 | 招生发送培训合同 | ✅ |
| 签约-学员确认 | 学员确认或主管代确认 | ✅ |
| 签约-自动创建简历 | 签约后自动创建 worker(pending) | ✅ |
| 管理员审核简历 | 审核通过后 worker=available | ✅ |
| 报名课程 | 招生代报名 / 主管代报名 / 学员自助报名 | ✅ |
| 讲师考核 | 按课程统一打分（合格/不合格） | ✅ |
| 转简历上户 | 合格学员转为可接单阿姨 | ✅ |
| 跳过培训转简历 | 签约后直接转简历 | ✅ |
